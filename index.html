⚠️ Немецкий голос не найден
Установите немецкий язык (Deutsch) в настройках синтеза речи.
📱 Android
1. Настройки → Язык и ввод → Синтез речи
2. Google TTS → Язык → добавить Deutsch
🍎 iPhone/iPad
1. Настройки → Универсальный доступ → Голоса
2. Добавьте голос Deutsch → Anna
💻 Windows
1. Параметры → Время и язык → Добавить язык Deutsch
2. Отметьте «Синтез речи», перезапустите Chrome
